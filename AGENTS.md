## Orchestrator - Program Manager
- Skill: `task-orchestration`
- Sorumluluk: Taskları parçalamak, önceliklendirmek, atamak ve ilerlemeyi izlemek.
- Sorumluluk: `TASKS.md`, `LOG.md`, `CHANGELOG.md`, `MEMORY.md` güncellemelerini koordine etmek.

## Core Builder - Userbot Engineer
- Skill: `polymarket-trading` (yakın alan: trading execution/risk workflow)
- Sorumluluk: Telethon listener, parser entegrasyonu, approval akışı ve CCXT executor kodunu geliştirmek.

## QA Planner - Test Engineer
- Skill: `integration-test-planner` + `unit-test-starter`
- Sorumluluk: Entegrasyon test planı ve unit test kapsamını üretmek.

## Security Guardian - Security Engineer
- Skill: `security-quick-scan` + `config-hardening` + `threat-modeling`
- Sorumluluk: Konfigürasyon ve runtime güvenlik kontrollerini gözden geçirmek.

## Memory Keeper - Process Analyst
- Skill: `task-orchestration`
- Sorumluluk: Hataları analiz edip `MEMORY.md` (local/global) guardrail güncellemelerini yönetmek.
